# Tougaloo Concierge Brand

- Primary colors: Tougaloo red and royal blue.
- Academic/student-support mascot: Eagle.
- Athletic mascot: Bulldog.

The Nest Student Concierge is an academic and student-support experience, so it uses the Eagle identity. Bulldog branding should be reserved for athletics-specific experiences and should not replace the Eagle within the Concierge.

The current interface uses representative red and royal-blue digital values. Replace them with institutionally approved hexadecimal brand values when the official brand guide is provided.
